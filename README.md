## Setup

### Development

```
npm i
npm run dev
```

### Deployment

```
安裝 Git
安裝 Nodejs 18.6.xx
安裝 pm2 
npm install pm2 -g
npm i
設置環境變數
npm run build
pm2 start
pm2 startup
pm2 save
```

### Check pm2 process 

```
pm2 ls
```
